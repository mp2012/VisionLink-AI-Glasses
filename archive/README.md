# VisionLink 历史迭代版本

本目录保留了项目从原型到最终架构的完整演化路径，仅供追溯参考。

## PC 端迭代 (Windows/Linux)

| 文件 | 说明 |
|------|------|
| `main.py` | 最早的 PC Demo 原型，仅 3 种模式（场景描述/文字识别/物体检测） |
| `8.py` ~ `16.py` | PC 端逐步迭代版本，逐步增加功能模式和交互优化 |

## Jetson 端迭代

| 文件 | 说明 |
|------|------|
| `Jetson_1.py` ~ `Jetson_4.py` | Jetson Orin Nano 边缘部署的迭代版本，经历量化调优和硬件适配 |

## 演化路径

```
main.py (PC Demo, 3 模式)
  └→ 8.py ~ 16.py (PC 端功能扩展)
      └→ Jetson_1.py ~ Jetson_4.py (边缘端移植)
          └→ src/ + apps/ (模块化重构, 当前架构)
```

## 当前架构

当前项目已完全重构为模块化架构，核心代码位于 `src/`，应用入口位于 `apps/`。本目录中的文件不再维护，仅保留作为项目演进记录。
